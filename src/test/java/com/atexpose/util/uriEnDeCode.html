URI encode decode test page

This page is a util page to use with unit test UrlCodingTest to test server side uri encoding and decoding. As there oddly enough exists no java method to decode any of the three options to encode parameters in JavaScript(escape, encodeURI and encodeURIComponent), there is a proprietary implementation. But this requires JavaScript functions encodeURIComponent and decodeURIComponent.

So this page has two uses:
1) To generate string that are uriencoded in browser. Use unit test to decode them and test that the result string is the same used as input in the browser.

2) Use server side code to uri encode below large string. Paste the resulting string into variable 'encodedServerSide' and there should be a true output in the second text area.
